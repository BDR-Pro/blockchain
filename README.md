#### Setup Real Quick

Grab the goods with:

```bash
git clone https://github.com/BDR-Pro/blockchain.git
cd blockchain
```

Make sure your `Cargo.toml` is popping with:

```toml
chrono = "0.4"
serde = { version = "1.0", features = ["derive"] }
sha2 = "0.10"
openssl = "0.10"
serde_json = "1.0"
```

#### Let's Roll 🎲

Jump into your project space and kick it off with:

```bash
cargo run
```
